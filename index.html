Exercício Prático: Manipulação do DOM

Objetivo: Fazer com que a imagem principal mude quando um dos botões com miniaturas for clicado.

Sua Tarefa:

  1. No arquivo `script.js`, selecione a imagem principal (`id="imagemExibida"`).
  2. Selecione todos os botões que estão dentro da `div` com a classe `botoes-troca`.
  3. Para cada botão, adicione um ouvinte de evento de clique (`addEventListener`).
  4. Dentro da função do evento, você precisará pegar o valor do atributo `data-src` do botão que foi clicado.
  5. Por fim, atribua esse valor ao atributo `src` da imagem principal para que ela mude!
Todo o código JavaScript deve ser escrito no arquivo `script.js`. Abra o console (`F12`) para qualquer log que você queira adicionar.
